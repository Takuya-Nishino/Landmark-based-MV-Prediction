# Reproducibility Checklist

## Before running
- [ ] Python environment created
- [ ] `requirements.txt` installed
- [ ] Source Excel file placed locally
- [ ] Column names checked against script assumptions

## Before sharing
- [ ] No raw patient data committed
- [ ] No restricted facility metadata committed
- [ ] No local machine paths remain in edited files
- [ ] Generated outputs reviewed for identifiers
- [ ] README matches current workflow
- [ ] Manuscript version and repository version synchronized
